# 🚀 启动后端服务器 - 操作指南

## ⚠️ 重要提示

由于系统PowerShell执行策略限制，**请使用CMD（命令提示符）**来启动服务器。

---

## 📋 方法一：使用CMD（推荐）

### 步骤1：打开命令提示符

按 `Win + R` 键，输入 `cmd`，然后按回车。

### 步骤2：安装依赖

在CMD中依次输入以下命令：

```cmd
cd /d e:\download\course-ai-tutor-main\course-ai-tutor-backend
npm install
```

等待安装完成（大约1-2分钟）。

### 步骤3：启动服务器

```cmd
node src\server.js
```

### ✅ 成功标志

看到以下输出表示启动成功：

```
========================================
   🚀 Course AI Tutor - Backend Server
========================================
   环境：development
   服务地址：http://0.0.0.0:8081
   本地访问：http://localhost:8081
   API 前缀：/api
========================================
✅ 后端服务已启动
✅ 数据库已连接
✅ JWT 认证已启用
========================================
```

## 📋 方法二：双击批处理文件

### 步骤1：安装依赖

1. 按 `Win + R`
2. 输入 `cmd` 并回车
3. 运行：
   ```cmd
   cd /d e:\download\course-ai-tutor-main\course-ai-tutor-backend
   npm install
   ```

### 步骤2：启动服务器

直接双击这个文件：
```
e:\download\course-ai-tutor-main\course-ai-tutor-backend\start.bat
```

---

## 📋 方法三：使用文件资源管理器

### 步骤1：打开CMD并定位到目录

1. 打开文件资源管理器
2. 导航到：`e:\download\course-ai-tutor-main\course-ai-tutor-backend`
3. 在地址栏输入 `cmd` 并回车（这会在此目录打开CMD）

### 步骤2：安装和启动

```cmd
npm install
node src\server.js
```

## 🎯 启动前端（可选）

如果需要启动前端，打开**另一个CMD窗口**：

```cmd
cd /d e:\download\course-ai-tutor-main\course-ai-tutor-frontend
npm install
npm run dev
```

然后访问：`http://localhost:5173`

## ❓ 常见问题

### Q: 提示 "npm 不是内部或外部命令"
**A**: Node.js未安装或未添加到环境变量。请重新安装Node.js。

### Q: 提示 "端口8081已被占用"
**A**: 修改 `course-ai-tutor-backend\.env` 文件中的 `PORT` 为其他值，如 `PORT=3000`

### Q: 提示 "Cannot find module"
**A**: 依赖未安装成功，请重新运行 `npm install`
