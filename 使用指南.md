## 🧠 智能体联动系统

### 架构特点

#### 1. 中枢智能体 (Main Agent)
- **职责**：接收用户输入，解析意图，调度子智能体
- **位置**：`src/agents/MainAgent.js`

#### 2. 共享知识库 (Shared Knowledge Base)
- **职责**：存储用户信息、学习数据、事件总线
- **位置**：`src/agents/SharedKnowledgeBase.js`

#### 3. 思维链引擎 (Chain-of-Thought Engine)
- **职责**：定义推理路径和模块调用顺序
- **位置**：`src/agents/ChainOfThoughtEngine.js`

### 预设思维链路径

| 路径名称 | 用途 | 步骤数 |
|---------|------|--------|
| knowledge_teaching | 知识讲解 | 4步 |
| qa_answering | 智能问答 | 5步 |
| personalized_learning | 个性化学习 | 6步 |
| wrong_question_review | 错题复习 | 5步 |
| psychological_counseling | 心理辅导 | 4步 |

### 模块联动流程

```
用户登录 → 用户中心同步信息到共享知识库
   ↓
用户提问 → 中枢智能体解析意图
   ↓
选择思维链 → 执行推理路径
   ↓
调度子智能体 → 从共享知识库获取数据
   ↓
生成结果 → 更新学习记录
   ↓
辅助模块收集 → 错题、笔记存储
   ↓
个性化模块调整 → 动态优化学习方案
```

## 📡 API接口文档

### 智能体接口

#### 1. 统一请求入口
```http
POST /api/agent/request
Authorization: Bearer {token}
Content-Type: application/json

{
  "type": "teaching",
  "content": "讲解Spring Boot依赖注入",
  "knowledgeBaseId": "kb_001"
}
```

#### 2. 聊天接口
```http
POST /api/agent/chat
Authorization: Bearer {token}
Content-Type: application/json

{
  "message": "你好，我想学习Java",
  "messageType": "text"
}
```

#### 3. 获取智能体状态
```http
GET /api/agent/status
Authorization: Bearer {token}
```

#### 4. 获取思维链路径
```http
GET /api/agent/paths
Authorization: Bearer {token}
```

#### 5. 同步用户信息
```http
POST /api/agent/sync-user
Authorization: Bearer {token}
Content-Type: application/json

{
  "learningGoal": "掌握Spring Boot",
  "subjectPreferences": ["Java", "Spring"]
}
```

### 功能模块接口

| 模块 | 接口 | 方法 |
|------|------|------|
| 知识点 | `/api/knowledge` | GET/POST/PUT/DELETE |
| 笔记 | `/api/notes` | GET/POST/PUT/DELETE |
| 错题本 | `/api/wrong-questions` | GET/POST/PUT/DELETE |
| 学习提醒 | `/api/reminders` | GET/POST/PUT/DELETE |
| 学习进度 | `/api/progress` | GET/POST |
| 学习统计 | `/api/progress/stats` | GET |
| 学习建议 | `/api/progress/recommendations` | GET |

## ❓ 常见问题

### Q1: 404错误
**原因**：后端服务器未启动  
**解决**：运行 `启动服务器.bat` 或 `node src\server.js`

### Q2: 端口被占用
**解决**：修改 `.env` 文件中的 `PORT` 值
```env
PORT=3000
```

### Q3: 依赖安装失败
**解决**：
```cmd
cd e:\download\course-ai-tutor-main\course-ai-tutor-backend
npm cache clean --force
npm install
```

### Q4: 如何测试智能体系统
**解决**：
```bash
# 1. 获取智能体状态
curl http://localhost:8081/api/agent/status

# 2. 获取思维链路径
curl http://localhost:8081/api/agent/paths

# 3. 发送请求（需要token）
curl -X POST http://localhost:8081/api/agent/request \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d "{\"type\":\"teaching\",\"content\":\"讲解Java\"}"
```
